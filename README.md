# TextRail

#### 介绍
使用汇编语言在dosbox-x上实现的一款简单的文字游戏，其中的背景和设定借用了《崩坏：星穹铁道》

#### 运行环境

1. 通过VScode配置的dosbox-x MASM-v5.00环境
2. 使用GB2312编码（此编码可用双字节表示汉字以此避免打印中文出现乱码）
3. 开发语言：8086汇编语言


#### 使用说明

在vscode中配置好相关环境，打开此文件，右键选择运行即可
